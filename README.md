## Installation
Follow these steps to get the project up and running on your local machine:

1. **Install packages:**
    
    Make sure you have Node.js and npm installed.
    ```bash
    npm install
    ```

## Running the Project

To run the project locally:
```bash
npm run dev
```

Run prod site mode

```bash
npm build
```
